# AetherTodo — Безпечний локальний робочий стіл продуктивності

Сучасний, захищений та функціональний To-Do інтерфейс, в якому конфіденційність стоїть на першому місці. Створений на чистому HTML5, CSS та JavaScript, **AetherTodo** шифрує ваші завдання, нотатки та чек-листи безпосередньо в браузері за допомогою вбудованого Web Crypto API.

#### 🌐 Демонстраційна версія: [https://saikoful.github.io/todo/](https://saikoful.github.io/todo/)
#### Доступно мовами: [🇬🇧 English](README.md) | [🇷🇺 Russian](README_RU.md)

## Ключові можливості

### 🛡️ Безпека з нульовим розголошенням (Zero-Knowledge)
*   **Мастер-пароль**: Всі записи блокуються при виході. При першому запуску створюється мастер-пароль.
*   **Шифрування AES-GCM 256-біт**: Дані завдань, чек-листи, категорії та налаштування шифруються на льоту.
*   **Деривація ключа PBKDF2**: Ключі шифрування обчислюються локально з використанням алгоритму PBKDF2 (100,000 ітерацій SHA-256) та випадкової криптографічної солі.
*   **Таймер автоблокування**: Автоматично блокує інтерфейс та стирає розшифровані дані в пам'яті при неактивності користувача (налаштування: 1хв, 5хв, 15хв, 30хв або ніколи).
*   **Повна автономність**: Жодних серверів та хмар. Пароль та дані ніколи не залишають ваш пристрій.

### 🍅 Віджети продуктивності
*   **Таймер Pomodoro**: Інтегрований таймер з круговою SVG-шкалою. Режими: Робота (25 хв), Коротка перерва (5 хв) та Довга перерва (15 хв). Статистика циклів зберігається в зашифрованій базі.
*   **Панель аналітики**: Динамічний підрахунок виконаних, поточних та загальних завдань з радіальним кільцем прогресу.
*   **Віджет погоди**: Використовує відкритий API Open-Meteo (пошук за назвою міста + поточна погода). Якщо ви поза мережею, автоматично симулює реалістичні погодні умови за локальним часом.
*   **Цитата дня**: Виводить мотивуючі фрази через сторонній API з локальним списком розробницьких афоризмів для резервної копії.

### 🎵 Дизайн та аудіо-інтеграція
*   **Естетичний аудіоплеєр**: Відтворює фонову музику (`files/song.mp3`). Вінілова платівка (з використанням `files/a.gif`) обертається в такт відтворенню, а звукові хвилі візуалізують програвання.
*   **Висувний регулятор гучності**: Повзунок гучності компактно прихований та плавно висувається при наведенні курсору на іконку звуку.
*   **Синтез аудіо-дзвінків**: За допомогою технології **Web Audio API** генерує м'які переливи при завершенні завдань та таймерів Pomodoro, не потребуючи завантаження важких аудіофайлів.
*   **Скляний інтерфейс (Glassmorphism)**: Стильний дизайн панелей з ефектом розмиття заднього фону (backdrop blur), плавними мікро-анімаціями кнопок та дрейфуючими кольоровими сферами на фоні.

### 📝 Просунуте керування завданнями (CRUD)
*   **Компактна форма введення**: Деталі завдання (дедлайн, пріоритет, категорія) приховані під анімованою кнопкою «Advanced Options», оптимізуючи робочий простір.
*   **Нотатки та чек-листи**: Картки розкриваються, дозволяючи писати довгі коментарі (автозбереження після розфокусування) та відзначати чек-поінти.
*   **Розумний пошук, фільтри та сортування**: Миттєвий пошук за назвами та описами; розділення на всі/активні/виконані; сортування за датою додавання, дедлайном або вагою пріоритету.
*   **Шифрований експорт/імпорт**: Вивантаження всієї бази у вигляді зашифрованого JSON-файлу (для імпорту потрібно буде ввести пароль) або в текстовому форматі.

## Технологічний стек
*   **Розмітка**: Семантичний HTML5
*   **Стилізація**: CSS3 (CSS Variables, Glassmorphism, Backdrop Filters, Keyframe Animations)
*   **Логіка**: ES6+ JavaScript
*   **Шифрування**: Web Crypto API (`crypto.subtle`)
*   **Іконки**: Lucide Icons CDN
*   **Шрифти**: Google Fonts (Outfit & Inter)

## Встановлення та запуск

1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/saikoful/todo.git
   cd todo
   ```
2. Запустіть будь-який локальний статичний сервер:
   ```bash
   # Через Python
   python3 -m http.server 8080
   
   # Через Node.js
   npx http-server -p 8080
   ```
3. Відкрийте `http://localhost:8080` у веб-браузері.
